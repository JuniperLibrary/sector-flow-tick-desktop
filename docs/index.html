关于

Sector Flow Tick 是一个基于 Electron 的桌面应用，直接请求东方财富板块资金流接口，提供实时资金流向监控、板块排行榜、走势图等功能。无需后端服务，开箱即用。

功能特性

轻量、实时、可配置的板块资金流监控工具

📡

实时采集

直接请求东方财富板块资金流接口，支持 1/3/5 分钟采集频率

📊

板块排行

展示最新板块快照，支持涨跌幅、净流入、换手率等多字段排序

📈

走势简图

点击板块查看资金流走势曲线，直观了解资金变化趋势

🎯

自定义采集

支持行业/概念/地域板块切换，自由勾选关注的采集板块

🌙

深色模式

跟随系统主题自动切换深色/浅色模式

独立运行
